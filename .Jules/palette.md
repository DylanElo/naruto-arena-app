## 2024-05-22 - Counter Builder Accessibility
**Learning:** Interactive lists implemented as `div`s with `onClick` exclude keyboard users.
**Action:** Always use `<button type="button">` for interactive list items, using `text-left` and `w-full` to maintain block layout.
